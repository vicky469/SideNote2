# Release Highlights

This release expands local tag workflows in the note sidebar and improves responsiveness across sidebar and source-note editing.

## Highlights

- Add local tag browsing and batch tag actions in the note sidebar, including cleaner batch-state reset behavior.
- Speed up sidebar edit, save, delete, and resolve flows by avoiding redundant local data refreshes and reducing thread signature work.
- Improve source markdown responsiveness by making editor highlight range calculation cheaper during typing and selection changes.
- Tighten the tag-tab toolbar layout so controls remain visible in narrower sidebars and search fields no longer crowd out buttons.

## Notes

- This release ships as `2.0.53` because `2.0.52` already exists as a GitHub tag/release.
