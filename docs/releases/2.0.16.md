# Release Highlights

This release fixes a Thought Trail rendering regression and tightens the plugin against the current Obsidian review checks.

## Highlights

- Restored Thought Trail Mermaid styling so connected-file graphs render correctly instead of collapsing into black shapes.
- Switched the repo to the official Obsidian ESLint plugin and cleaned up the remaining source issues around deprecated APIs, DOM rendering, and async overrides.
- Hardened several sidebar and support surfaces while keeping build, lint, and test coverage green.

## Notes

- There are no note-format or vault-data migrations in this release.
