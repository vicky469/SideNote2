# Release Highlights

This release tightens the index toolbar layout so the mode tabs and filter controls fit more cleanly in the sidebar.

## Highlights

- Made the index mode tabs more compact and rebalanced their width so `List`, `Thought Trail`, and `Agent` use the row more cleanly.
- Switched the `Files` and `Resolved` toolbar filters to icon-only controls so the toolbar matches the rest of the row and wastes less space.
- Reduced extra padding in the tab control so the sidebar toolbar stays denser without changing the overall visual style.
