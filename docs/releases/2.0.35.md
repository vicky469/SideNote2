# Release Highlights

This release smooths out sidebar card interactions so editing, controls, and source reveals feel more predictable.

## Highlights

- Comment action buttons now stay hidden until a card is explicitly selected, instead of appearing on hover or scroll.
- Threads with nested replies keep their show or hide toggle visible at all times, so it remains obvious where to click.
- Inline editing no longer steals the caret when you reposition it inside the draft textarea.
- Revealing an anchored side note now scrolls to the actual stored text span, which fixes long-line notes where the selected text could still be off-screen.

## Notes

- There are no note-format or vault-data migrations in this release.
