# Release Highlights

This release restores the note sidebar’s Thought Trail header controls so users can switch back to list mode after opening Thought Trail.

## Highlights

- Ensure the individual note sidebar always renders its toolbar in Thought Trail mode, including list/thought-trail tab controls.
- Keep the note sidebar header visible for the mode switch, preventing users from being stuck in Thought Trail.
- Syncs correctly after rebuilding and installing the updated bundle.
