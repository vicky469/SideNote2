# Release Highlights

This release adds a faster deleted-comments cleanup flow in the page sidebar and keeps the deleted view scoped to the current note.

## Highlights

- Added `Empty trash` in the page sidebar deleted view to permanently clear deleted side notes for the current source note.
- Removed the extra confirmation prompt for `Empty trash` so deleted-view cleanup is one step.
- Kept the deleted-comments cleanup logic out of the index sidebar and covered the file-scoped purge flow with new tests.

## Notes

- `Empty trash` only appears in the page sidebar while deleted view is active.
- The action clears deleted threads and deleted child entries for the current source note, then returns the sidebar to the normal view.
