# Release Highlights

This release makes the sidebar easier to scan by surfacing the note context first and shrinking timestamp noise.

## Highlights

- Anchored side notes in the note sidebar now show the selected text before the timestamp, with ellipsis when the preview is too long.
- Index sidebar rows now lead with the most useful context: page notes show the page name, while anchored notes show the selected text.
- Sidebar timestamps now use a more compact Obsidian-style format such as `1:30 PM`, `Yesterday`, `Fri 1:30 PM`, or `Mar 10`.
- Persisted sidebar cards no longer react visually on hover while scrolling; the neutral active state and action icons now appear when the card is selected.

## Notes

- These changes are presentation-only. They do not change comment storage or thread data.
