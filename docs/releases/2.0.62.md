# Release Highlights

This release tightens comment editing behavior and sidebar tab state styling.

## Highlights

- Comment editor formatting now handles bold and highlight operations more consistently for multiline and list-item selections.
- Comment draft dismissal is less aggressive when interacting with sidebar comments, so normal sidebar clicks do not unexpectedly exit the active edit flow.
- Sidebar tab states now follow Obsidian theme text colors, avoid hardcoded black, and make disabled tabs visibly faded.
- Project philosophy documentation has been refreshed.

## Notes

- There are no note-format or vault-data migrations in this release.
