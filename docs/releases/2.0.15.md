# Release Highlights

This release tightens the inline draft editor so adding a side note stays compact and typing-focused in narrow desktop sidebars.

## Highlights

- Removed the explicit draft textarea label that was surfacing as a distracting accessibility popup during customer-facing add-comment flows.
- Fixed the focused draft editor so typing uses the real textarea layer instead of a misaligned overlay preview.
- Reduced the default add/append draft height and removed the narrow-screen textarea inflation that made placeholders and typed text appear oversized.

## Notes

- There are no note-format or vault-data migrations in this release.
