# Release Highlights

This release makes persisted side note cards quieter by default so the display stays focused on the note itself.

## Highlights

- Hid comment action icons, author labels, and footer chrome by default on hover-capable devices, then revealed them on hover or active focus.
- Removed the `You` author badge from user-written persisted comments to cut visual noise.
- Kept agent reply author labels visible when needed and preserved touch-device usability by leaving controls visible on non-hover devices.

## Notes

- This change only affects persisted display cards. Edit mode and streaming cards keep their existing behavior.
